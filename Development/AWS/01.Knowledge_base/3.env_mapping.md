# Environment Mapping: Local vs. Cloud

| Feature             | Local Dev (Laptop)                   | Production (AWS)                       | The "Bridge"                       |
| ------------------- | ------------------------------------ | -------------------------------------- | ---------------------------------- |
| **Compute**         | Running `npm start` in terminal      | **EC2** or **Lambda**                  | Git (Deploys code)                 |* 
| **Database**        | Docker Container (`postgres:alpine`) | **Amazon RDS** (Managed Postgres)      | Connection String (`DATABASE_URL`) |
| **Secrets**         | `.env.local` file                    | **AWS Secrets Manager**                | Environment Variables              |
| **Networking**      | `localhost:3000`                     | **Route 53** + **Nginx/Load Balancer** | DNS A Records                      |
| **Logs**            | `console.log`                        | **Amazon CloudWatch**                  | AWS IAM Role                       |
| **Scheduled Tasks** | `cron` / `setTimeout`                | **Amazon EventBridge**                 | Trigger Rules                      |
| **Traffic Rules**   | None (Mac Firewall)                  | **Security Groups**                    | Inbound/Outbound Rules             |
