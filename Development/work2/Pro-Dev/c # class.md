Abstraction allows making relevant information visible 

encapsulation enables a programmer to *implement the desired level of abstraction*
**Encapsulation** is defined 'as the process of enclosing one or more items within a physical or logical package'.

Access specifiers - state access rules for class + members.  
 - class default:  **internal**. 
- members default:  **private**.

- Encapsulation is implemented by using **access specifiers**. An **access specifier** defines the scope and visibility of a class member. C# supports the following access specifiers −
* Public - Any public member can be accessed from outside the class.
* Private
* Protected
* Internal
* Protected internal
 

Data type -  type of variable, and return type specifies the data type of the data the method returns, if any.

dot (.) operator - used to access members / connects class +members
