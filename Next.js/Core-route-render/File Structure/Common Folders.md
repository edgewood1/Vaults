

**Common Folders**
 * /public: Holds static assets that are served directly from the root URL (e.g., public/logo.png is available at /logo.png).  These are images, fonts, etc.   They are served from root URL (/)
 * /src: (Optional) An alternative to the root, many developers place their app, pages, components, etc., inside src for a cleaner project root.
 * /components or /app/ui: (Convention) A common folder for storing reusable React components (buttons, cards, forms).
 * /libs or /app/lib: (Convention) A place for utility functions, data-fetching logic, and other helper code that isn't a React component.
 * /hooks: (Convention) A folder for custom React hooks.
 * /styles: (Convention) A folder for global stylesheets, if not using app/ui/global.css.
