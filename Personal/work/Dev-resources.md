## Git

```bash
git reset --hard        # changes all tracked files to match most recent commit
git clean -f            # removes untracked files
git add <file>          # staging — creates a snapshot
```

*Core concept: working directory → staged snapshot → committed snapshot*

**Revert:** creates commit Z that mirrors X (non-destructive)
