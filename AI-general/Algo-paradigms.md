

The **learning algorithms** are the specific **step-by-step procedures, mathematical methods, or computational processes** that are used to actually _perform_ the learning _within_ one of these paradigms. They are the "recipes" used to find the patterns or tune the parameters of a chosen model architecture.   

**Examples Linking Paradigms and Algorithms:**

- **Paradigm:** Supervised Learning
    - **Model Architecture Chosen:** Linear Regression, Logistic Regression, Neural Network   
        
    - **Common Learning Algorithm:** **Gradient Descent** (and its variations like SGD, Adam) is often used to find the best parameters (numbers) for these architectures.   
        
    - **Model Architecture Chosen:** Decision Tree
    - **Common Learning Algorithm:** Algorithms like **CART** or **ID3** are used to decide how to build the tree structure.   
        
- **Paradigm:** Unsupervised Learning
    - **Goal/Model Type:** Clustering
    - **Common Learning Algorithm:** **K-Means algorithm** or **DBSCAN algorithm**.
- **Paradigm:** Reinforcement Learning
    
    - **Model Architecture Chosen:** Policy Network (often a Neural Network)
    - **Common Learning Algorithm:** **Q-Learning algorithm**, **SARSA algorithm**, **Policy Gradient methods**.
